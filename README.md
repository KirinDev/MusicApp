# MusicApp
 A music or any audio player developed in java and javaFX;
 Can play any audio as long is in 'wav' format;
 The files are stored in a music directory;
 App has a authentication system of email and password;
 User can register an account;
 Implemented with a Microsoft SQL server database;
App funcionalities:
- play any audio that is in the system;
- pause the audio and resume it at the respetive time of the pause;
- stop the audio completely;
- exit to the next or previous audio;
- can create playlists and add audios(musics) to it;
- adjust the volume of the audio;
